# @glasshome/ui

SolidJS component library for GlassHome, built on Kobalte.

## Install

```bash
bun add @glasshome/ui
```

## Quick Start

```tsx
import { Button, Card, Badge } from "@glasshome/ui";

function App() {
  return (
    <Card>
      <Badge>New</Badge>
      <Button onClick={() => console.log("clicked")}>Click me</Button>
    </Card>
  );
}
```
